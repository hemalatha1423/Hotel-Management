# Hotel Management System

A Python + MySQL based Hotel Management System developed to manage hotel operations such as hotel details, staff records, rooms, guests, bookings, and payments.

## 📌 Features

- Add, Update, Delete, Display Hotels
- Manage Staff Details
- Manage Room Types and Rooms
- Guest Registration
- Booking Management
- Payment Management
- Menu Driven Console Application

## 🛠️ Technologies Used

- Python
- MySQL
- mysql.connector

## 📂 Database Tables

- HOTEL
- STAFF
- ROOMTYPE
- ROOM
- GUEST
- BOOKING
- PAYMENT
